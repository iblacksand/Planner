[back](http://iblacksand.github.io/advanced-planner/) #List of Commands ### Run Run is used to execute any command that you can do in your command line To run a program do ``` run notepad ``` replace notepad with the name of the program. This can work with any command that you can do with command prompt or equivalence. ### Display **This is not currently fully implemented** Display is used to display the text given. This will currently only show on the terminal currently with [release v0.15](https://github.com/iblacksand/advanced-planner/releases/tag/v0.15). it is just used simply as ``` display hello world ``` this will display hello world ### Loop Loop is used like so: ``` loop 5m 5 { run notepad display hello world } ``` This will loop 5 times with 5 minutes in between each loop.
