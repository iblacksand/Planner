What is Advanced-Planner?

  • Advanced Planner is a way to make your terminal a better tool to help be productive during your day.

  • You can easily space and timeout your different console commands for cmd prompt and bash(not tested, yet)

  • It makes it easy for people to store scripts in a simple to edit .txt with simple formatting

What can it do?

Advanced-Planner can space, repeat and allow tracking of commands using the command prompt, or whatever it is started in.

It relies on just 4 commands. The biggest and most useful being run. This will run commands after it. For example:

run notepad

This does have limitations due to windows being horrible at opening things in cmd.

Other commands

Advanced-Planner also has the loop command. This is in the syntax:

loop time amount

It also features display which is as simple as it sounds

Example:

loop 2s 5
{
display Starting NotePad
run notepad
}

Other features

Advanced-Planner features properties, which is a tag that can be attached to any command to have a delay from the command before, or what ever you put

Unfortunately, right now it only supports delays

run.prop(5s)

This would delay run for 5 seconds.

It also has alias which will allow you to run another advanced-planner script.

alias script/script.txt

this would run script.txt when it went to that line

The Compiler (well sort of)

While it is not technically a compiler, it does change the input code into a more simplified version. The compiler will check your syntax and will make a report for you.

Issues and Complications

The biggest issue for me is that lack of properties that would make things easier, as well as no support for linux. This project has made me grow towards linux without even using it.

Properties are not compilable(although it won't break anything) and delaying is the only function they have.

Another issue is that the way I close the window doesn't work on some computers(such as the school computers) but it seems to work on all updated computers so there is not much I can do.
