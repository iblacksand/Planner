[back](/advanced-planner) # **THERE IS NO RELEASE OUT YET** # Advanced Planner ## Build automatically 1. Download the latest [release](https://github.com/iblacksand/advanced-planner/releases) 2. Start run-build.bat 3. Put the argument that you want to have for the command then click the command 4. Repeat until you fully build your script 5. Click on finish and put main.txt in the text box that pops up 6. Click on compile and check if you have any errors and if you do then restart from step 2 7. Click on run and you have successfully ran the program. ## Build manually 1. Download the latest [release](https://github.com/iblacksand/advanced-planner/releases) 2. open main.txt and put in your script 3. Start run-man.bat 4. put in the text box main.txt and click compile 5. If you have errors then edit the script according to what errors are shown 6. Click run if you have no errors
